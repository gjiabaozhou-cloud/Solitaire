# Tkinter 52-card Deck Viewer

Simple Tkinter app that displays a 52-card deck in a grid. Features:

- Shuffle the deck
- Reset to ordered deck
- Deal the top card

Run the GUI:

```bash
python3 show_deck.py
```

Run tests (headless):

```bash
python3 test_deck.py
```

No external packages required (uses standard library `tkinter`).

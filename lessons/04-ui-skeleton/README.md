# Lesson 04 — UI Skeleton (tkinter viewer)

- **Subject:** Introduction to GUI basics with `tkinter` — windows, drawing simple card rectangles and labels.
- **Core concepts:** event loop, widgets, canvas drawing, basic callbacks.
- **How to explain to kids:** Compare the program to a puppet stage — the code tells the stage what to draw and when. Show how clicking can be handled.
- **Activities:** Open `show_deck.py`. Run it and observe cards drawn. Change card colors or label text and re-run to see immediate visual feedback.
- **Checks:** Can they change a label on a card? Can they wire a button that shuffles and redraws the deck?

```bash
python3 show_deck.py
```

- **Estimated time:** 40–60 minutes.

Exercises:
- Change the grid columns and card size.
- Add a "Shuffle" button that reloads the grid using `shuffle_deck`.

**Concept Mapping:** See [teaching overview](../../teaching/overview.md) for MVC mapping and how view/controller interact with model functions.

# Lesson 01 — Deck Basics

- **Subject:** Cards and deck data — representing playing cards as simple data.
- **Core concepts:** tuples/records, lists, iteration, mapping.
- **How to explain to kids:** Compare a card to a "postcard" with two labels: `rank` (number/face) and `suit` (shape). A deck is a stack of postcards. Show a physical deck if possible.
- **Activities:** Inspect `deck.py`; print a single card, print the whole deck; write a function to show `rank` and `suit` as text.
- **Checks:** Can they list 52 unique cards? Can they write a small snippet that prints "Ace of Spades"? Run:

```bash
python3 run.py
```
- **Estimated time:** 20–30 minutes.

Exercises:
- Modify `RANKS` to add a Joker and update `generate_deck()`.
- Print only red suits.

**Concept Mapping:** See [teaching overview](../../teaching/overview.md) for how this lesson teaches lists and tuples and where it fits in the broader curriculum.

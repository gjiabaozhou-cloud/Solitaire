# Lesson 13 — Final Play, Tests & Packaging

- **Subject:** End-to-end playable demo, final tests, README and packaging notes for students/teachers.
- **Core concepts:** combining earlier lessons, writing simple README instructions, packaging as a runnable folder/script, minimal test harness.
- **How to explain to kids:** This is the "show" — package everything so a friend can run the game. Encourage writing a short "how to play" paragraph.
- **Activities:** Create the final folder with the polished UI connected to the final model, include `README.md` with run steps, and add a small test script confirming startup and a few basic moves. Optionally create a record of the student's play (screenshot).
- **Checks:** Does `python lessons/13-final-play/run.py` (or the included run script) launch the game? Do included tests pass? Can a user follow the `README.md` to run and play?

```bash
cd lessons/13-final-play
# python3 run.py
```
